# Ray Tracing In One Weekend

My Implementation (Rust) of RTIOWE ([_Ray Tracing in One
Weekend_](https://raytracing.github.io/books/RayTracingInOneWeekend.html))

### What is this

A simple ray-tracer using raylib for visualization.
